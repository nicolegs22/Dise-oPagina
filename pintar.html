Cuadro Semántico: Header

Este cuadro está pintado de color **lightblue** y contiene el título de la página.

Cuadro Semántico: Nav

Color de fondo: **lightgray**. Se utiliza para contener enlaces de navegación.

Cuadro Semántico: Main

Color de fondo: **lightgreen**. Aquí se presenta el contenido principal del documento.

Cuadro Semántico: Section 1

Color de fondo: **lightyellow**. Esta sección contiene información relacionada con el contenido.

Cuadro Semántico: Section 2

Color de fondo: **lightyellow**. Otra sección que agrupa contenido relevante.

Cuadro Semántico: Article

Color de fondo: **lightpink**. Este cuadro contiene un artículo o contenido independiente.

Cuadro Semántico: Aside

Color de fondo: **lightcoral**. Se utiliza para mostrar contenido relacionado, como una barra lateral.

Cuadro Semántico: Footer

Color de fondo: **lightblue**. Contiene el pie de página con información adicional o de contacto.
